Deskband

Privacy Policy
✓ No data collection · No tracking · No servers

Overview

Deskband is a browser companion character that animates alongside your browsing activity. Your privacy is fully protected — the extension operates entirely on your own device and never communicates with any external server.

What data is stored

Deskband stores the following information locally on your device using chrome.storage.local:

  • Keystroke count (today, yesterday, all-time best)
  • Mouse click count (today, yesterday)
  • XP points and level
  • Character position on screen
  • Current date (for daily stat rollover)
This data exists only on your device. It is never transmitted, uploaded, or shared with anyone — including the developer.

What data is NOT collected

Deskband does not collect, store, or transmit:

  • The content of any keystrokes (only the count)
  • Your browsing history or visited URLs
  • Any personally identifiable information
  • Location data
  • Any data about the websites you visit

Third parties

Deskband does not share any data with third parties. There are no analytics services, advertising networks, or external APIs involved. The extension contains no remote code — all logic runs locally from the installed package.

Why all URLs?

Deskband uses the <all_urls> permission so the companion character can appear on every page you visit. This permission is used only to inject the visual overlay — it is not used to read, access, or record any page content.

Changes to this policy

If this privacy policy ever changes, the updated version will be posted at this URL. The extension will be updated in the Chrome Web Store accordingly.
